CSS variables

现有的CSS预处理器都包含了变量等概念，CSS也不甘落后，变量的特性相信在不久的未来也会为广大浏览器厂商所接受。

Example:

// Declare
.class {
  --bg-color: #f00;
}
// Use
.container {
  background-color: var(--bg-color);
}
