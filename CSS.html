CSS supports

  • Basic use
@supports (display: flex) {
    div { display: flex; }
}
  • not关键字
@supports not (display: flex) {
    div { float: left; } /* alternative styles */
}
  • Logic operations
/* or */
@supports (display: -webkit-flex) or
          (display: -moz-flex) or
          (display: flex) {

    /* use styles here */
}

/* and */
@supports (display: flex) and (-webkit-appearance: caret) {
	
	/* something crazy here */
}
  • JavaScript support

绑定在window上的全局对象CSS:

var supportsFlex = CSS.supports("display", "flex");

CSS variables

现有的CSS预处理器都包含了变量等概念，CSS也不甘落后，变量的特性相信在不久的未来也会为广大浏览器厂商所接受。

Example:

// Declare
.class {
  --bg-color: #f00;
}
// Use
.container {
  background-color: var(--bg-color);
}
