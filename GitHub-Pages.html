GitHub Pages

Introduction

GitHub Pages是GitHub推出的托管静态页面的解决办法。

一般有两种GitHub Pages类型：

  1. 如你的用户名为foo，那么与foo.github.io这个二级域名同名的GitHub repository对应的master分支上的代码将出现在你的「用户」的GitHub Page上。

  2. 一般的项目中建立的gh-pages分支对应一个GitHub Pages。而其对应的URL为：foo.github.io/repo-name。
