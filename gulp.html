gulp

Official site: gulp.com

Install

npm install gulp -g

Install plugins

Find plugins here.

e.g.

npm install gulp-concat --save-dev

Example

Sample gulpfile.js:

var gulp = require('gulp');
var uglify = require('gulp-uglify');
 
gulp.task('compress', function() {
  gulp.src('lib/*.js')
    .pipe(uglify())
    .pipe(gulp.dest('dist'))
});

then run:

gulp compress
