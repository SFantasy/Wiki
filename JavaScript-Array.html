JavaScript Array

Define

使用字面量声明数组

// Good
var arr = [];
// Bad
var arr = Array();

Common methods

  • join and toString

将数组转换为字符串

var names = ['Tom', 'Jelly', 'Jerry'];
var nameStr = names.join();
  • concat and splice 
var a = [1, 2, 3];
var b = [4, 5, 6];
a.concat(b);
// [1, 2, 3, 4, 5, 6]

splice可以取数组里从某下标开始的几个元素组成一个新的数组，当然也可以向数组的任意位置插入新的元素

a.splice(0, 1);
// [1]
a.splice(0, 0, 1);
// [1, 2, 3 ...]
  • push and unshift

向数组添加元素

var a = [2, 3, 4];
a.push(5);
a.unshift(1);
// return the length of the array
  • pop and shift

取数组元素

var a = [1, 2, 3];
a.pop();
// 3
a.shift();
// 1
  • reverse and sort 
var a = [1, 2, 3];
a.reverse();
// [3, 2, 1]

sort

String:

var a = ['bca', 'abc', 'bac', 'cba'];
a.sort();
// [ 'abc', 'bac', 'bca', 'cba' ]

Number:

var nums = [3, 5, 1, 2];
nums.sort(function (a, b) {
  return a - b;
});

Iterations

forEach

var x = [1, 2, 3];
x.forEach(function (num) {
  console.log(num * num);
});

every and some

参数为一个返回布尔值的函数：

var x = [1, 2, 3];
var test = x.every(function (num) {
  return num > 0;
});

var test2 = x.some(function (num) {
  return num > 0;
});

reduce

JavaScript中还有reduceRight方法，执行顺序和reduce相反

var nums = [1, 2, 3, 4, 5];
nums.reduce(function (sum, num) {
  return sum + num;
});

map

类似reduce，但是会生成一个新的数组

filter

类似every，但是不返回布尔值，而是返回应用过后的新数组
